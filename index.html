Az if-eknél, ha 8E-t ír oké, ha 8e-t akkor nem. Megoldható, hogy ne legyen érzékeny a kis-nagybetűre? Igen, teljesen megoldható! 😄 A JavaScript-ben az egyszerű módja az, hogy kisbetűsre vagy nagybetűsre alakítod a felhasználó által beírt értéket az if-ek előtt, így a “8E” és a “8e” ugyanaz lesz. A te kódodban például így módosíthatod az openLine() függvényt: function openLine() { let line = document.getElementById('line').value.trim().toUpperCase(); // <-- ide tettem a .toUpperCase() let date = document.getElementById('date').value; if(line === "8E") line = "0085"; if(line === "20E") line = "0205"; if(line === "25") line = "0250"; if(line === "225") line = "2250"; if(line === "30") line = "0300"; if(line === "30A") line = "0301"; if(line === "230") line = "2300"; if(line === "31") line = "0310"; if(line === "35") line = "0350"; if(line === "36") line = "0360"; if(line === "44") line = "0440"; if(line === "53") line = "0530"; if(line === "66") line = "0660"; if(line === "66B") line = "0660"; if(!line || !date) { alert("Kérlek töltsd ki mindkét mezőt!"); return; } let url = `https://online.winmenetrend.hu/budapest/latest/lines/${line}/vehicles?date=${date}`; window.open(url, "_blank"); } ✅ Amit csináltunk: A document.getElementById('line').value.trim().toUpperCase() biztosítja, hogy minden input nagybetűs legyen, így a “8e”, “8E”, “8e”, “8E” mind ugyanazt eredményezi. Nem kell minden if-hez külön kis-nagybetű változatot írni. Ha akarod, készíthetek neked egy teljes frissített verziót a teljes HTML-edhez, ami: Enterre működik Mobilbarát Kis-nagybetű érzéketlen a vonalaknál Akarod, hogy elkészítsem? if(line === "25") line = "0250"; - Van or? Igen, lehet több feltételt egyszerre kezelni az if-ben az OR (||) operátorral. 😄 Például a te sorodat így lehet “or”-ral írni, ha két lehetséges input ugyanazt az értéket adja: if(line === "25" || line === "025") line = "0250"; Az || azt jelenti: “ha az egyik vagy a másik igaz”. Tehát ha a felhasználó beírja “25” vagy “025”, a line értéke “0250” lesz. Ha a cél kis-nagybetűt is figyelmen kívül hagyni, akkor előtte érdemes a line-t nagybetűsre alakítani: let line = document.getElementById('line').value.trim().toUpperCase(); if(line === "8E" || line === "08E") line = "0085"; if(line === "25") line = "0250"; // itt nem kell kis-nagybetű, mert szám Ha akarod, tudok készíteni egy “OR és kis-nagybetű érzéketlen” verziót a teljes openLine() függvényedhez, hogy minden vonalat rugalmasan kezeljen a rendszer.
