# WinMenetrend URL Generátor

Ez a HTML + JavaScript kód a WinMenetrend Online oldalához generál URL-eket a gyors és könnyű keresés érdekében.
https://online.winmenetrend.hu/

**Fontos:**  
- Nem gyűjt, nem tárol és nem továbbít semmilyen személyes adatot.  
- Belső és/vagy nyilvános használatra is biztonságos.  

**Használat:**  
1. Nyisd meg a `index.html` fájlt böngészőben.  
2. A megfelelő adatokat beírva gyorskeresést biztosít a kód.
